 **Project Description:-

The Student Management System is a web-based application developed using Python and Django framework. This project is designed to simplify the process of managing student information in an educational institution. It provides an efficient way to store, update, view, and manage student records in a centralized system.

The application helps administrators maintain student details such as personal information, academic records, and other relevant data. By digitizing student management processes, the system reduces manual paperwork, improves data accuracy, and saves time.

** Features:-

- Add new student records
- View student details
- Update student information
- Delete student records
- User-friendly interface
- Secure data storage using SQLite database

** Technologies Used:-

- Python
- Django
- SQLite
- HTML
- CSS

**Purpose of the Project:-

The main purpose of this project is to provide an easy-to-use platform for managing student data efficiently. It demonstrates the use of Django for web development, database management, and CRUD (Create, Read, Update, Delete) operations in a real-world application.
